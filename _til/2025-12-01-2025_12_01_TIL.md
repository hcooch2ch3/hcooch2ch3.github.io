# TIL(Today I Learned)

12월 1일(일)

**공부한 내용을 제 언어로 빠르게 정리한 내용이라서 잘못된 내용이 있을 수 있습니다.**

**Flutter**
- AnimatedDefaultTextStyle
    - 텍스트 스타일이 변경될 때 자동으로 애니메이션을 적용하는 위젯
    - duration 속성으로 애니메이션 시간을 지정하고, style 속성이 변경되면 부드럽게 전환
    - 색상, 크기, 폰트 등 TextStyle의 모든 속성 변화에 애니메이션 적용 가능
- import 에러 발생했을때 코드액션으로 Import문 자동 추가 가능
    - 빨간 밑줄이 있는 곳에 커서를 두고 Quick Fix (Cmd + .)를 실행
- StatefulWidget의 State 내부의 메서드는 context를 어디서든 사용가능
    - State 클래스는 BuildContext를 프로퍼티로 가지고 있어 별도 파라미터 없이 접근 가능
    - build 메서드뿐만 아니라 initState, dispose 등 모든 State 메서드에서 this.context로 사용 가능
