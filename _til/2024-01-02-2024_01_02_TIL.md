## 서버 스터디

- pnpm: npm과 같은 패키지 매니저
- nextjs: 리액트 뷰가 서버 사이드 렌더링 되도록 하는 프레임워크
  - react와의 차이?

    - 클라이언트 브라우저에서 뷰가 조립됨.
- tailwind: css 프레임워크
- nginx: 웹 서버 소프트웨어
- Monorepo: 하나의 코드 저장소에서 여러 프로젝트를 관리하는 방식
