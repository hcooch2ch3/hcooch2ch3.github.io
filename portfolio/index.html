“남은 또 다른 나라고 생각하고 결과를 만듭니다.”

내가 불편한건 남도 불편하고, 내가 싫은건 남도 싫어합니다. “다른 사용자”가 제가 구현한 UI를 쓴다면 직관적이고 편리할까를 고민하고, “다른 팀원”이 제 코드와 문서를 본다면 이해하기 쉬울까를 고민합니다. “남”의 시선으로 “나”를 바라볼 수 있을때 스스로 개선해야할 점이 보이고, 스스로 개선할 수 있을때 지속적인 성장이 가능하다고 생각합니다. 끊임없이 스스로 되묻고 더 나은 결과물을 만들기 위해 노력합니다.

퓨처키친: Kitchen Automation Platform

퓨처키친의 자사 치킨 브랜드인 ‘What a CR!SP’ 매장 치킨 로봇을 작동시키기 위한 사용자 인터페이스를 개발하였습니다. 작업 입력, 로봇 설정, 조리 상태 체크, 치킨 픽업 완료, 인벤토리 입력 기능을 제공하여 사용자가 치킨 로봇 구동을 가능하게 합니다. 사용기술은 React.js / Django 입니다.

퓨처키친: 코코이찌방야 주방 작업 지시 시스템

코코이찌방야 본사 매장 조리 작업 지시 시스템의 사용자 인터페이스를 개발하였습니다. 주문이 들어오면 시스템이 자동으로 인식하여 해당 파트별 필요한 작업 내용을 보여주는 시스템입니다. 사용기술은 React.js / Django 입니다.

퓨처키친: What a CR!SP iOS APP

‘What a CR!SP’ 의 치킨을 주문할 수 있는 iOS 앱입니다. 메뉴 소개, 장바구니 기능, 결제 기능, 마일리지 기능, 리뷰 작성 및 이미지 편집 기능, 리뷰 좋아요 기능 등을 개발하였습니다. 사용 기술은 SwiftUI, Combine 입니다.

반올림 서비스

해외 직구 전문 셀러들을 대상으로, 상품 소싱을 클릭 한번으로 가능하게 하고 소싱한 상품 목록 엑셀 파일을 자동으로 생성 해주는 서비스를 개발하였습니다. 관리자는 원하는 상품 페이지 URL을 등록하면 백엔드에서 해당 URL의 상품 정보를 크롤링 하여 상품 목록 DB에 저장합니다. 소싱 상품 목록 페이지 구현, 엑셀 파일 생성 기능, 상품 페이지 크롤링 기능 구현을 주로 담당하였습니다.
